### UI Philosophy
- **Matte over Gloss**: Reduced glow intensity for a more premium, grounded feel.
- **Organic Depth**: Gradients are subtle, mimicking deep forest lighting rather than neon cyber-punk.
- **Roundness**: Large border-radii (`rounded-[32px]`) for a friendly yet structured look.
- **Shadows**: Soft, colored shadows (`box-shadow: 0 0 20px rgba(5, 150, 105, 0.25)`) to create lift without harshness.
